# Glossary

- **Agentic app**: Application that invokes MCP tools to complete tasks.
- **Bulkhead**: Resource isolation to prevent one domain from starving others.
- **Delta query**: Graph pattern for incremental sync using delta tokens.
- **Entra ID**: Microsoft identity platform (formerly Azure AD).
- **Idempotency key**: Unique token for safe retries on write operations.
- **MCP**: Model Context Protocol, a standardized interface for tools.
- **MCP client**: The caller that invokes tools on an MCP server.
- **MCP server**: Service that implements MCP tools and policy enforcement.
- **PKCE**: Proof Key for Code Exchange used with OAuth2 auth code flow.
- **Rate limit**: Policy controls for per-user or per-tenant request volume.
- **Retry-After**: Header indicating when to retry after throttling.
- **Tenant**: Microsoft 365 organization boundary (directory).
- **Token broker**: Server component that stores and refreshes tokens on behalf of users.
